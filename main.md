# Legacy Spring Project
This project is an example of how to use an old version of spring

## Tools
- JDK 8
- Spring 2.0.9.RELEASE
    - spring-data
- Mysql 8.0.17
- Lombock 
- Maven 3.8.6

## Table
only one table was used, with this sql format

| Campo | Tipo                |
|-------|---------------------|
| id    | bigint(20) unsigned |
| name  | varchar(100)        |
